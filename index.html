🔥 실시간 동기화 설정

Firebase를 설정하면 모든 브라우저에서 동일한 상품을 실시간으로 보고 관리할 수 있습니다!

🔥 Firebase 설정 방법:

  1. 1. Firebase Console에 접속
  2. 2. "프로젝트 추가" → 프로젝트 이름 입력
  3. 3. "웹 앱 추가" → 설정 코드 복사
  4. 4. "Firestore Database" → "데이터베이스 만들기" → "테스트 모드"
  5. 5. 위 양식에 붙여넣기 → "Firebase 연결" 클릭
🎯 동기화 해결책:
• Firebase 연결하면 모든 브라우저에서 실시간 동기화
• 상품 등록/수정/삭제가 즉시 전 세계에 반영
• 다른 브라우저에서도 동일한 상품 목록 확인 가능
